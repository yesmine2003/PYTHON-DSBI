# 🐍 Python Learning Project - From Fundamentals to Data Analysis

## 📖 About the Project
This repository gathers a collection of Jupyter notebooks documenting my learning journey in Python. The goal of this project is to practically explore data structures, basic algorithms, as well as the initial steps of data manipulation (Data Science).

## 🚀 Skills and Concepts Covered

The project is structured as progressive "Challenges":

### 1. Variables, Types, and Operations
*   Discovery, manipulation, and verification of different variable types (`int`, `float`, `str`, `bool`)[cite: 5].
*   Use of mathematical operations (power, modulo, division) and the `math` library for calculations and rounding (`floor`, `ceil`)[cite: 6].
*   Exploration of bitwise operators[cite: 6].

### 2. Data Structures (Collections)
*   **Tuples:** Creation, indexing, slicing, and concatenation[cite: 4]. Highlighting the immutability of this structure[cite: 4, 5].
*   **Lists:** Creation, adding elements (`append`), reversing order (`reverse`), managing nested lists, and calculating sums[cite: 5].
*   **Strings:** Recombining words into sentences, text cleaning, and splitting to prepare for textual data analysis[cite: 6].
*   **Sets:** Generation of random lists via the `random` library, conversion to a `set` to remove duplicates, and application of mathematical operations on sets (union, intersection, difference, subset verification)[cite: 7].
*   **Dictionaries:** Creation and data access via unique keys[cite: 5]. Advanced sorting algorithms applied to dictionaries (sorting by keys and values using `operator.itemgetter`) to analyze word frequency in a song[cite: 8].

### 3. Logic and Algorithms (Conditions & Loops)
*   Implementation of conditional structures (`if`/`elif`/`else`) and loops (`for`/`while`) for problem-solving[cite: 10].
*   Concrete application exercises: identifying numbers divisible by 3, converting month names to the number of days, reversing words, and variants of the classic "FizzBuzz" algorithm[cite: 10].

### 4. Introduction to Data Science (Pandas)
*   Workspace configuration (installation of `ydata_profiling`)[cite: 9].
*   Importing and reading databases from Excel files (`liquor_store_sales.xlsx`) using the `pandas` library (`read_excel`)[cite: 9].
*   Initial exploration of DataFrames: visualizing the first rows with `df.head()`, extracting specific columns, and using `df.describe()` to get descriptive statistics[cite: 9].

## 🛠️ Technologies and Libraries Used
*   **Language:** Python 3
*   **Environment:** Jupyter Notebook
*   **Standard Libraries:** `math`, `random`, `operator`[cite: 6, 7, 8].
*   **Data Science:** `pandas`, `ydata_profiling`[cite: 9].
